# AM VPN

A secure and fast VPN client for Android with both free and premium servers worldwide.

## Features

- Free and Premium servers worldwide
- Real-time speed monitoring
- Server load balancing
- Secure VPN protocols
- Beautiful material design UI
- Easy server switching
- Connection statistics

## Server Locations

### Free Servers
- North America: US (New York, Los Angeles), Canada (Toronto)
- Europe: UK (London), Germany (Frankfurt), Netherlands (Amsterdam)
- Asia: Japan (Tokyo), Singapore, India (Mumbai)
- Others: Australia (Sydney), Brazil (Sao Paulo)

### Premium Servers
- North America: Multiple locations in US and Canada
- Europe: UK, Germany, France, Netherlands
- Asia: Japan, Singapore, Hong Kong, South Korea, India
- Others: Australia, Brazil, South Africa

## Installation

1. Requirements:
   - Android Studio Arctic Fox or later
   - Android SDK 33 or later
   - JDK 8 or later
   - Android device running Android 5.0 (API 21) or later

2. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/am-vpn.git
   ```

3. Open in Android Studio:
   - Open Android Studio
   - Select "Open an Existing Project"
   - Navigate to the cloned repository
   - Click OK

4. Build and Run:
   - Connect an Android device or start an emulator
   - Click the "Run" button (green play icon)
   - Select your device
   - Wait for the app to install and launch

## Usage

1. Launch AM VPN
2. Select a server from the list
   - Free servers are marked with "AM VPN Free"
   - Premium servers are marked with "AM VPN Pro" and a lock icon
3. Click the connect button
4. Grant VPN permissions when prompted
5. Wait for connection to establish
6. Monitor your connection speed and statistics

## Premium Features
